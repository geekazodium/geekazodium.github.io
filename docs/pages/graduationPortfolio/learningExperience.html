The Love of Computers


computers have always been facsinating to me, this magical box being able to draw awsome looking graphics and do math at speeds no human can dream of. Later I realized that it's not magic, and as I learned more and more about computers, the more it has become a huge part of my life's meaning, to create something awsome with computers.

Leap of overconfidence: minecraft plugin development

I really liked minecraft, I REALLY liked minecraft, I play that game too much and even to this day I still play minecraft arguably too much. I have decided to try making a jump to making minecraft server plugins, this went... not super great, turns out, I still didn't understand more advanced programming concepts such as OOP or even understand what an "instance" is, it forced me to realize I didn't know enough to make anything more than code that spawns tnt at the player when they right click. While this didn't give me much to show in terms of making a final product, that's not what matters, because it helped me learn Java and lead up to my next big step.

Heck: minecraft modification development

(during grade 9) This was a much bigger leap, I thought at the time, I understood coding well and was already very good at it, but I was wrong. I decided to learn minecraft modification writing using FabricMC, which was, unbeknownced to me, a heck of a challenge, while these two tasks seem similar they were, and I must EMPHASIZE (haha get it, the html tag is <em>) that it was very different.

Plugin development with bukkit:
  • simple api
  • event listener model
  • indirect access to minecraft through api

Modification development with FabricMC:
  • you have to learn to read minecraft source code
  • you must interact with minecraft code directly
  • mixin(code injection) model

in summary, the fact that I thought I could manage that without any struggle was a really dumb idea. I spent way too long trying to make any minecraft mod, and the most I managed was some small projects (better lang reader) that are on my github right now. This took me a LONG time to learn and while at many moments painful, and a reminder that I should have been more prepared, I still think that I would do it the same way if I needed to learn coding again. As learning OOP concepts on the fly by reading minecraft code was, in my very dumb opinion, acutally really helpful for me.

(I later learned Minecraft Forge modding as well, which was much easier to learn because it was more like plugin development and I also already had experience with modding which helped me build a funny project)

the struggle may not have been always fun, but I cared about learning coding and didn't give up, in the end it definitely was worth it.
